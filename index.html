The Voyager Record

This is a demonstration of in-browser live decoding of the voyager record. This page was created by Malte Gruber. The source code can be found here.

Description

This page loads a .mp3 file that was created by reading a copy of the Voyager phonograph record, the .mp3 recording can also be found here. This page then runs some code in your browser that translates the audio recording of the record to an image (I.e what you see is decoded directly from the .mp3 audio file). The phonograph record has two channels, left and right. These channels hold different sets of images which are shown in the left and right windows. A time domain representation of the signal is shown as an oscilloscope graph, it matches the newest line of the image.
